# Codex 对话管家 macOS MVP

**产品名：** Codex 对话管家
**平台：** macOS
**形式：** 轻量原生 SwiftUI 应用
**定位：** Codex 聊天记录转换和基础管理工具。

## 核心目标

用户打开应用后，第一眼就能完成两件事：

- `API → 官方`
- `官方 → API`

会话内容不作为主功能展示。默认只读取列表和状态，避免用户本机会话很多时卡顿。

## 当前范围

- 自动识别本机 Codex 记录目录。
- 启动时快速读取本机会话列表。
- 固定中文三栏界面，避免系统侧栏隐藏/显示导致顶部布局跳动。
- 按全部、API、官方、异常筛选会话。
- 区分桌面会话和 CLI 会话。
- 按项目目录分组显示会话标题和相对时间。
- 支持单击勾选多条会话后批量转换。
- 未勾选时默认转移当前范围内的全部可转移会话。
- 勾选后只转移已选的 1 条、2 条或更多会话。
- 勾选后可批量删除；点开详情后可删除单条会话。
- 需要时检查本机记录。
- 转换前自动备份。
- 删除前自动备份，并从本机列表和记录中移除。
- Codex 运行中拒绝写入，避免影响进行中的任务。
- 恢复最近一次本地备份。

## 暂缓

- 完整聊天内容浏览。
- 备份历史选择器。
- 高级修复向导。
- 签名和 notarization。

## 安全规则

任何写入真实 Codex 记录前，必须先创建备份，并确认 Codex 已完全退出。转换必须保持本机列表和原始记录一致。

图片多的会话只改必要归属信息，不展开、不重写图片内容，降低大文件内存占用和内容损坏风险。
